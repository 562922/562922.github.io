Lessons

The proccess of building a website can be split into 4 main stages: Plan, Build, and Publish. During the planning stage you plan out the website by making a site map and picking a coding environment. In the build stage you actually work on the website starting from a simple webpage and moving all the way to a fully dynamic website with javascript interactables. And in the publish phase I will show you how to publish your site for free using github pages.
