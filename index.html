The Goal:

To help people learn to code a website from scratch.

My goal is to make an easy to understand learning platform that people can use and benifit from.

What you will find.

In this website you will be able to find different tools and learning assets that will hopefully help you on your journey.

  • In the Tips page you will find a list of common issues and fixes.
  • In the Help page you will find a guide on any and all tags that you will be using for any webpage or site.
  • In the Gallery page you will find images of code discribing the type of code and where it is used. As well as the images used in any given page and where we got it from.
